✓ Exploitability: TRIVIAL (just need multiple IPs)
✓ Impact: UNLIMITED code generation per wallet

Attack Vector:
  1. Attacker controls multiple IP addresses (VPN, proxies, botnet)
  2. Sends requests with SAME wallet from different IPs
  3. Each IP gets 5 requests per minute
  4. Total: N × 5 codes per minute (N = number of IPs)

Exploitation Potential:

  - 1,000 IPs: 5,000 codes/min = 7,200,000 codes/day
